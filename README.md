# LightsOut Visual для Oxygen Not Included

Мод добавляет визуальную темноту и свет от экипировки (скафандры, шлемы, маски). Не влияет на геймплейные дебаффы.

## 🔧 Сборка из исходников
Запусти build.bat — он создаст NewLightsOut.dll в папке мода.
## 🐛 Известные проблемы
На мульти-мониторных сетапах с разной герцовкой возможны редкие фризы. Решение: полноэкранный эксклюзивный режим или одинаковая герцовка.

## ⚙️ Настройка

Открой `Config.json` и поменяй параметры:

```json
{
  "Enabled": true,           // Вкл/выкл мод
  "DarknessBase": 0.15,      // Сила затемнения (0.0 - 1.0)
  "EquipLightBonus": 6.0     // Доп. радиус света от экипировки (метры)
}
```
## Примеры конфигов:
1. Мягкая атмосфера: "DarknessBase": 0.12, "EquipLightBonus": 5.0
2. Хардкор: "DarknessBase": 0.05, "EquipLightBonus": 2.0
3. Для скриншотов: "DarknessBase": 0.3, "EquipLightBonus": 8.0
